# 주피터 노트북 실행 방법 (VS Code)

이 저장소의 수업 파일(`.ipynb`)을 **VS Code**에서 여는 방법을 정리한다.

> 먼저 알아둘 점: 수업 노트북은 파이썬 기본 기능(`enum`, `dataclasses`, `random`)만 쓴다.
> 따로 설치할 라이브러리가 없고, 파이썬 환경과 VS Code만 있으면 된다.

---

## 1. VS Code 확장 프로그램 설치 (처음 한 번만)

VS Code 왼쪽의 확장(Extensions) 탭에서 아래 두 개를 설치한다.

- **Python** (Microsoft)
- **Jupyter** (Microsoft)

---

## 2. 수업용 conda 환경 만들기 (처음 한 번만)

conda는 이미 쓸 수 있는 상태라고 가정한다.
터미널(VS Code 안의 터미널이나 Anaconda Prompt)에서:

```bash
conda create -n pokemon python=3.11 ipykernel -y
```

- `-n pokemon` : 환경 이름 (원하는 이름으로 바꿔도 된다)
- `ipykernel` : VS Code가 이 환경으로 노트북을 실행하기 위해 필요하다

> 한 번 만들어 두면 다음부터는 이 단계를 건너뛴다.

---

## 3. 노트북 열고 커널 고르기

1. VS Code에서 `.ipynb` 파일을 연다. (탐색기에서 더블클릭)
2. 오른쪽 위 **커널 선택**(Select Kernel)을 누른다.
3. **Python Environments**에서 방금 만든 `pokemon` 환경을 고른다.

> 커널이 `pokemon` 으로 잡혀 있어야 노트북이 그 환경에서 실행된다.

---

## 4. 셀 실행하기

- `Shift + Enter` : 실행하고 다음 칸으로 이동
- `Ctrl + Enter` : 실행하고 그 자리에 머무름
- 셀 왼쪽의 ▶ 버튼을 눌러도 된다.
- **위에서부터 차례대로** 실행한다. 윗 칸을 건너뛰면 아래 칸에서 오류가 날 수 있다.
- 맨 위 툴바의 **Run All** 을 누르면 전체를 한 번에 실행한다.

---

## 5. 수업 파일 안내

각 주차 폴더 안에 두 파일이 있다.

- `..._student.ipynb` : 빈칸을 채우며 직접 푸는 **실습용**
- `..._answer.ipynb` : 완성된 **정답**

먼저 실습용으로 풀어 보고, 막히면 정답을 확인한다.

---

## 자주 묻는 것

- **커널 목록에 `pokemon` 이 안 보여요**
  → 2번에서 `ipykernel` 을 설치했는지 확인한다. 그래도 없으면 VS Code를 껐다 켠 뒤 다시 **커널 선택**을 눌러 본다.

- **`ModuleNotFoundError` 가 떠요**
  → 오른쪽 위 커널이 `pokemon` 으로 잡혀 있는지 확인한다. 다른 환경이면 바꿔 준다.

- **변수가 꼬였어요 / 실행 번호가 뒤죽박죽이에요**
  → 툴바의 **Restart**(커널 재시작) 후 **Run All** 로 처음부터 다시 돌린다.
