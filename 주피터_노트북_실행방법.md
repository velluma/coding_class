# 주피터 노트북 실행 방법

이 저장소의 수업 파일(`.ipynb`)을 여는 방법을 정리한다.

> 먼저 알아둘 점: 수업 노트북은 파이썬 기본 기능(`enum`, `dataclasses`, `random`)만 쓴다.
> 따로 설치할 라이브러리가 없고, **파이썬과 주피터만** 있으면 된다.

---

## 1. 수업용 conda 환경 만들기 (처음 한 번만)

conda는 이미 쓸 수 있는 상태라고 가정한다.
터미널(Anaconda Prompt 또는 PowerShell)에서:

```bash
conda create -n pokemon python=3.11 jupyterlab -y
```

- `-n pokemon` : 환경 이름 (원하는 이름으로 바꿔도 된다)
- `jupyterlab` : 노트북을 실행하는 프로그램

> 한 번 만들어 두면 다음부터는 이 단계를 건너뛴다.

---

## 2. 환경 켜기 (수업할 때마다)

```bash
conda activate pokemon
```

줄 앞에 `(pokemon)` 이 보이면 성공이다. 수업을 시작할 때마다 이 명령으로 환경을 켠다.

---

## 3. 노트북 열기

두 방법 중 편한 쪽을 고른다.

### 방법 A — 주피터랩 (브라우저)

수업 파일이 있는 폴더로 이동한 뒤 실행한다.

```bash
cd 파일이_있는_폴더경로
jupyter lab
```

브라우저가 열리면 왼쪽 목록에서 `.ipynb` 파일을 더블클릭한다.

### 방법 B — VS Code

1. VS Code에서 `.ipynb` 파일을 연다.
2. 오른쪽 위 **커널 선택**을 눌러 방금 만든 `pokemon` 환경을 고른다.
3. 확장 프로그램 **Python**, **Jupyter** 가 필요하다. 없다면 설치한다.

---

## 4. 셀 실행하기

- `Shift + Enter` : 실행하고 다음 칸으로 이동
- `Ctrl + Enter` : 실행하고 그 자리에 머무름
- **위에서부터 차례대로** 실행한다. 윗 칸을 건너뛰면 아래 칸에서 오류가 날 수 있다.

---

## 5. 수업 파일 안내

각 주차 폴더 안에 두 파일이 있다.

- `..._student.ipynb` : 빈칸을 채우며 직접 푸는 **실습용**
- `..._answer.ipynb` : 완성된 **정답**

먼저 실습용으로 풀어 보고, 막히면 정답을 확인한다.

---

## 자주 묻는 것

- **`ModuleNotFoundError` 가 떠요**
  → 환경을 켰는지(`conda activate pokemon`), VS Code라면 커널을 `pokemon` 으로 골랐는지 확인한다.

- **셀 실행 번호가 뒤죽박죽이에요 / 변수가 꼬였어요**
  → 메뉴의 **Restart & Run All**(커널 재시작 후 전체 실행)로 처음부터 다시 돌린다.

- **`jupyter` 명령을 못 찾는대요**
  → 환경을 켠 상태인지 확인한다. 안 켰다면 1번 환경에 `jupyterlab` 이 설치됐는지 확인한다.
